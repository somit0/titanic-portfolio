# Titanic Survival Prediction
A beginner-friendly project performing EDA and Logistic Regression on the Titanic dataset.


## Steps
1. Environment setup with Anaconda
2. Data cleaning and visualization
3. Logistic Regression model training
4. Model evaluation (accuracy, confusion matrix)


## Results
- Accuracy: 0.79
- Key insight: Females had a much higher survival rate than males.


## Tools Used
Python | Pandas | Matplotlib | Seaborn | Scikit-learn | Jupyter
